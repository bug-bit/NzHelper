## 功能特性

| 功能 | 说明 |
|------|------|
| ⏱ 计时记录 | 精确记录每次时长，支持后台计时 |
| 🏷 标签管理 | 自定义分类，让记录更加清晰 |
| 🔒 隐私保护 | 手势锁 + 指纹/人脸解锁 |
| 🗑 回收站 | 支持恢复误删除的数据 |
| 📊 数据统计 | 周报、月报趋势分析 |
| 🔔 通知提醒 | 实时显示计时状态 |
| 🎨 Material You | Jetpack Compose + Material 3 设计 |
| ⚡ 轻量高效 | 本地存储，小巧省电 |

## 下载安装

### APK 下载

前往 [GitHub Releases](https://github.com/bug-bit/NzHelper/releases) 下载最新版 APK。

- 支持系统：Android 8.1 (API 27) 及以上
- ABI：arm64-v8a / armeabi-v7a / x86 / x86_64

### 自行构建

```bash
git clone https://github.com/bug-bit/NzHelper.git
cd NzHelper
./gradlew assembleRelease
```

---

## 使用说明

1. 设置应用锁，保护个人隐私
2. 点击开始按钮记录过程
3. 完成后保存记录，可添加标签和备注
4. 在历史页面查看统计数据
5. 使用标签管理整理记录

> 所有数据均保存在本地，不上传服务器。
