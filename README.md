## 功能特性

| 功能 | 说明 |
|------|------|
| 计时记录 | 精确记录每次手艺活的时长，支持后台计时，再也不用担心被打断忘了时间 |
| 标签管理 | 自定义标签分类——晨撸、午冲、夜战，科学归纳你的战斗日志 |
| 隐私保护 | 手势锁 + 指纹/人脸解锁，防止社死级尴尬场面 |
| 回收站 | 手滑删错了？回收站给你一次改过自新的机会 |
| 统计图表 | 周报月报趋势图，看看这周是不是有点过度了 |
| 通知提醒 | 战斗中实时通知，沉浸式体验不忘看时间 |
| Material You | 采用 Jetpack Compose + Material 3 设计，撸也要撸得优雅 |
| 轻量高效 | 小包体，不占空间，不卡后台，省电又省心 |

## 下载安装

### 直接下载 APK

前往 [GitHub Releases](https://github.com/bug-bit/NzHelper/releases) 下载最新 APK 安装包。

- 支持系统：Android 8.0 (API 26) 及以上
- ABI 支持：arm64-v8a / armeabi-v7a / x86 / x86_64

### 自行构建

```bash
git clone https://github.com/bug-bit/NzHelper.git
cd NzHelper
./gradlew assembleRelease
```

## 使用指南

1. 打开 App，先设个手势锁 —— 别让借你手机的人发现你的秘密
2. 点击「开始」—— 即开始记录手艺活的时间
3. 完事了点「结束」—— 加上标签和备注，记录今天手感如何
4. 主页查看历史记录和趋势图 —— 知己知彼，百战不殆
5. 标签管理 —— 给每场战斗一个名分
6. 手滑删了？去回收站捞回来

> 注意：所有数据仅存本地，不上传云端，你的秘密只有你自己知道
